Flavour Hunting


Tired of fruitlessly searching for rock melon ice cream on foot, I set up a script to do the legwork for me! 13 lines of python sets up a web scraper targeting the menus of my local stores, checks if the menus contain the sacred words “rock melon”, then Pushbullet API takes care of sending push notifications to my phone. 

import urllib2
from pushbullet import Pushbullet

api_key = "KEY"
pb = Pushbullet(api_key)
my_icecream_channel = pb.channels[0]

page = urllib2.urlopen("menu url").read()
Flavour_list = ['Rock Melon', 'rock melon', 'Rock melon', 'Rockmelon', 'rockmelon']
for key in Flavour_list:
    if key in page:
        push = my_icecream_channel.push_note("ICECREAM NOTIFICATION",
                                "There is %s at the shop!" % key)

Future iterations may replace Pushbullet with a SMS or email based approach as Pushbullet can be rather buggy
