Flask and foolproof databases


A friend from a inter-university association approached me to migrate their current spreadsheet based records system to a hosted database with easy to use search tools.

The solution is a SQLAlchemy database built around Flask, all hosted on Heroku to minimise costs and quicken delivery. In order to be a sustainable solution, the database needs to be able to be updated with with ease. Pyexcel and Flask-Excel are used to allow users to upload spreadsheets of new entries and automatically append the database, as well as generate and download spreadsheets of filtered results.

This is a lightweight, zero cost solution that should provide an opportunity to see if this solution is a fit for the association without substantial investment.
